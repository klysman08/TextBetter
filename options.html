How it works

Select text on any webpage or double-click to select a word. A floating TextBetter action bar will appear immediately.

Choose an action (like "Make it professional") to rewrite the text. You can then copy it or insert it directly back into input fields.

Prompt Customization

Customize the instruction templates sent to Gemini for each action.

TextBetter AI Writing Assistant. All API calls are executed directly from your browser to Google Gemini API servers.
